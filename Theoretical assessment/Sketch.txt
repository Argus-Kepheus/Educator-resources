Você está trabalhando com um script LaTeX para geração automática de provas educacionais. O script possui as seguintes características principais:

### Funcionalidades Atuais:
- **Sistema de gabarito**: Controle via `\setboolean{showanswers}{true/false}` para gerar versões para professor (com gabarito) e aluno (sem gabarito)
- **Numeração automática**: Seções numeradas com algarismos romanos (I, II, III, etc.) e perguntas numeradas sequencialmente
- **Formatação em duas colunas**: Layout otimizado para impressão
- **Equações inline**: Uso de `$...$` para expressões matemáticas
- **Alternativas com marcação**: Sistema de quadrados vazios (□) para alunos e preenchidos (■) para gabarito

### Estrutura dos Comandos:
- `\titulosecao{Nome da Seção}` - Cria seções temáticas com numeração romana automática
- `\pergunta` - Cria perguntas numeradas automaticamente
- `\alternativa{Texto da alternativa}{correta}` - Cria alternativas (usar "correta" para alternativa correta, deixar vazio para incorreta)

  \ordem(4)={4}

    \rand % Gera número aleatório
    \pgfmathparse{int(mod(\arabic{rand}, \arabic{tempcount}) + 1)}
    \let\randpos\pgfmathresult
    
    % Troca posições
    \checkord(\arabic{tempcount})
    \let\temp\cachedata
    \ordem(\arabic{tempcount})={\ordem(\randpos)}
    \ordem(\randpos)={\temp}

  }

\newcommand{\embaralharalternativas}[8]{%

  \alternativas(2)={#2}

  \alternativas(4)={#4}

    % Gera ordem aleatória

    % Mostra alternativas na ordem embaralhada

    \whiledo{\value{altloop}<5}{%

      \let\pos\cachedata

      \stepcounter{altloop}
